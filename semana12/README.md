#### Labeddit

#### O que funciona
Login,
Cadastro,
Telas Protegidas,
Loading,
Carregamento de Posts,
Criar Post
Votação de Post (p/cima, p/baixo, ou anular),
Comentários, 
Criar Comentários,
Votação de Comentário (p/cima, p/baixo, ou anular),
Desafio: Paginação

#### O que não funciona
Sem estilização,
Desafio: sem logout
Desafio: sem compartilhamento
